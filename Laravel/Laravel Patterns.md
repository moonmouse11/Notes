# Laravel Patterns
***
## Porto
Архитектурный паттерн в Laravel.
1. DDD (Domain Drive Development).
2. Модульность
3. MVC (Model View Controller)
4. Слои
5. ADR (Action Domain Responder)
Porto состоит из двух слоев:
- Корабль.
	- Родительские классы.
	- Автозагрузка компонентов.
	- Функции и классы.
- Контейнер.
