# Dockerfile
***
## Basic
**Dockerfile** - файл с инструкциями для создания docker image.
***
## Dockerfile Directives

``` Dockerfile
FROM        # Указание базового образа.
ADD	        # Добавляет локальные и удаленные файлы и директории.
ARG	        # Объявление аргумента.
CMD	        # Команда для запуска контейнера.
COPY	    # Копирует файлы и директории.
ENTRYPOINT	# Предустановленная команда при запуске образа.
ENV	        # Идентификатор переменной. Вызывается со знаком $.
EXPOSE	    # Устанавливает порт для образа
FROM	    # Указание базового образа.
HEALTHCHECK	# Проверка целостности контейнера перед запуском.
LABEL	    # Добавляет метаданные в образ.
MAINTAINER	# Устанавливает автора образа.
ONBUILD	    # Устанавливает инструкции, для выполнения во время сборки образа.
RUN	        # Выполнение команды и фиксация нового слоя для образа.
SHELL	    # Устанавливает командную оболочку по умолчанию для образа.
STOPSIGNAL	# Устанвливает сигнал для завершения работы образа.
USER	    # Устанавливает пользователя и группу.
VOLUME	    # Объявление директории для обмена файлами между контейнерами.
WORKDIR	    # Установка рабочей директории контейнера.
```
***
## Example
``` Dockerfile
FROM node:latest

LABEL org.label-schema.version=v1.1

ENV NODE_ENV="dev"

ENV PORT=3000

RUN mkdir -p /var/node

ADD src/ /var/node/

WORKDIR /var/node

RUN npm install

EXPOSE $PORT

CMD ./bin/www
```
